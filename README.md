# PiBokeh_Plus
Using AI to add bokeh to images taken by PiCamera on Raspberry Pi 4.

Launch server: run server/main_server.py

Do local inference: run server/main_server.py

Do online inference: launch server, change the ip address in server/main_server.py and run server/main_server.py --online=True
